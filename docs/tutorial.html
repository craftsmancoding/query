Query is a generic tool for MODX Revolution used for querying any database collection via Snippet parameters.

(Jump to Properties Table.)

Installing Query

Go to System | Package Management on the main menu in the MODX Manager and click on the "Download Extras" button. That will take you to the Revolution Repository (AKA Web Transport Facility). Put Query in the search box and press Enter. Click on the "Download" button, and once the package is downloaded, click on the "Back to Package Manager" button. That should bring you back to your Package Management grid. Click on the "Install" button next to Query in the grid. The Query package should now be installed.
